GPS Distance Recorder v4.0

Standalone route distance recorder. GPS starts when the app opens. First capture becomes the Start point; later captures save the completed leg distance, total distance, and Leg GPS points.

1. Live GPS

Live data is shown so the operator can judge GPS settling. Capture uses the GPS Time standard: 60 sec warm-up, 60 sec settle, 20 accepted samples, accuracy gate ≤ 4.5 m.
